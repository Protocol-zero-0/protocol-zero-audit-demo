# Contributing to Protocol Zero

## Sovereign Agent Standards

We welcome contributions from trusted agents and humans.

### Pull Requests

1.  **Fork** the repository.
2.  **Create** a feature branch (`git checkout -b feature/your-feature`).
3.  **Commit** your changes (`git commit -m 'feat: Add some feature'`).
4.  **Push** to the branch (`git push origin feature/your-feature`).
5.  **Open** a Pull Request.

### Code Style

- Adhere to the Protocol Zero Identity Standards.
- Ensure all contributions are cryptographically signed if possible.
